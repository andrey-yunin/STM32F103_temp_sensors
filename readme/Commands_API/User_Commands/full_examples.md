## Формат пакета и CRC

### Пакет команды (PC → Анализатор)

```
┌────────────┬──────────┬───────────┬──────────────┬──────┐
│ Header     │ Length   │ Command   │ Parameters   │ CRC  │
│ 3 байта    │ 2 байта  │ 2 байта   │ N байт       │ 1 б  │
│ 43 4D 3E   │ BE       │ BE        │              │      │
└────────────┴──────────┴───────────┴──────────────┴──────┘
```

- **Header**: всегда `0x43 0x4D 0x3E` (ASCII "CM>")
- **Length**: кол-во байт после поля Length (Command + Parameters + CRC)
- **CRC**: XOR всех байт от Command до последнего байта Parameters

### Пакет ответа (Анализатор → PC)

```
┌────────────┬──────────┬───────────┬──────┬──────────┬──────────┬──────┐
│ Header     │ Length   │ Command   │ Type │ Status   │ Data     │ CRC  │
│ 3 байта    │ 2 байта  │ 2 байта   │ 1 б  │ 2 байта  │ N байт   │ 1 б  │
│ 43 4D 3E   │ BE       │ эхо       │      │ 00 00=OK │          │      │
└────────────┴──────────┴───────────┴──────┴──────────┴──────────┴──────┘
```

- **Type**: `0x01`=ACK, `0x02`=DONE, `0x03`=DATA, `0x04`=ERROR
- **CRC**: XOR всех байт от Command до последнего байта Data

### Алгоритм CRC

```
CRC = Command[0] XOR Command[1] XOR Param[0] XOR Param[1] XOR ... XOR Param[N-1]
```

### Типы данных

| Тип    | Размер   | Порядок    | Примечание                    |
|--------|----------|------------|-------------------------------|
| UINT8  | 1 байт   | —          |                               |
| UINT16 | 2 байта  | Big-endian | Старший байт первый           |
| INT16  | 2 байта  | Big-endian | Дополнение до двух            |
| UINT32 | 4 байта  | Big-endian |                               |
| STRING | N байт   | ASCII      | Без нуль-терминатора          |

### Коды целей (target/source)

| Код  | Устройство       |
|------|------------------|
| 0x01 | REACTION_DISK    |
| 0x02 | REAGENT_ROTOR    |
| 0x03 | SAMPLE_DISK      |
| 0x04 | WASH_STATION     |
| 0x05 | WASTE            |

## 1. Системные команды (0x10xx)

### 1.1. GET_STATUS (0x1000)

**Описание**: Запрос текущего состояния анализатора.

**Параметры**: нет

**TX пакет**:
```
43 4D 3E 00 03 10 00 10
│         │     │     └── CRC: 0x10 XOR 0x00 = 0x10
│         │     └── Command: 0x1000 (GET_STATUS)
│         └── Length: 3 (2 cmd + 1 crc)
└── Header: "CM>"
```

**RX — ACK**:
```
43 4D 3E 00 06 10 00 01 00 00 11
                   │  │     └── CRC: 0x10^0x00^0x01^0x00^0x00 = 0x11
                   │  └── Status: 0x0000 (OK)
                   └── Type: 0x01 (ACK)
```

**RX — DATA** (статус: idle, ошибок нет):
```
43 4D 3E 00 09 10 00 03 00 00 00 00 00 13
                   │  │     │  │     └── CRC
                   │  │     │  └── error_code: 0x0000 (нет ошибки)
                   │  │     └── status: 0x00 (idle)
                   │  └── Status: 0x0000 (OK)
                   └── Type: 0x03 (DATA)
```

CRC ответа: `0x10^0x00^0x03^0x00^0x00^0x00^0x00^0x00 = 0x13`

### 1.3. INIT (0x1002)

**Описание**: Инициализация модулей анализатора.

**Параметры**: modules (UINT8) — битовая маска модулей, 0xFF = все

**Сценарий**: Инициализация всех модулей

**TX пакет**:
```
43 4D 3E 00 04 10 02 FF ED
│         │     │     │  └── CRC: 0x10^0x02^0xFF = 0xED
│         │     │     └── modules: 0xFF (все модули)
│         │     └── Command: 0x1002
│         └── Length: 4 (2 cmd + 1 param + 1 crc)
└── Header
```

**RX — ACK**:
```
43 4D 3E 00 06 10 02 01 00 00 13
```
CRC: `0x10^0x02^0x01^0x00^0x00 = 0x13`

**RX — DONE**:
```
43 4D 3E 00 06 10 02 02 00 00 10
```
CRC: `0x10^0x02^0x02^0x00^0x00 = 0x10`

**Описание**: Запрос версии прошивки анализатора.

```
                      └── CRC: 0x10^0x03 = 0x13
```

**RX — ACK**:
```

```

**RX — DATA** (версия 2.5, сборка 100, дата "2025-01-15"):
```
43 4D 3E 00 14 10 03 03 00 00 02 05 00 64 32 30 32 35 2D 30 31 2D 31 35 73
│         │     │     │  │     │  │  │     └── date: "2025-01-15" (ASCII)
│         │     │     │  │     │  │  └── build: 0x0064 = 100
│         │     │     │  │     │  └── minor: 0x05
│         │     │     │  │     └── major: 0x02
│         │     │     │  └── Status: 0x0000
│         │     │     └── Type: 0x03 (DATA)
│         │     └── Command: 0x1003
│         └── Length: 20 (2+1+2+14+1)
└── Header
```

CRC расчёт:
```
0x10^0x03 = 0x13  ^0x03 = 0x10  ^0x00 = 0x10  ^0x00 = 0x10
^0x02 = 0x12  ^0x05 = 0x17  ^0x00 = 0x17  ^0x64 = 0x73
^0x32('2') = 0x41  ^0x30('0') = 0x71  ^0x32('2') = 0x43  ^0x35('5') = 0x76
^0x2D('-') = 0x5B  ^0x30('0') = 0x6B  ^0x31('1') = 0x5A  ^0x2D('-') = 0x77
^0x31('1') = 0x46  ^0x35('5') = 0x73
```

### 1.5. SET_DATETIME (0x1004)

**Описание**: Установка даты и времени анализатора.

**Сценарий**: Установка 2025-01-15 14:30:45

**TX пакет**:
```
43 4D 3E 00 0A 10 04 07 E9 01 0F 0E 1E 2D C9
│         │     │     │     │  │  │  │  │  └── CRC
│         │     │     │     │  │  │  │  └── second: 0x2D = 45

│         │     │     │     │  │  └── hour: 0x0E = 14
│         │     │     │     │  └── day: 0x0F = 15
│         │     │     │     └── month: 0x01 = 1
│         │     │     └── year: 0x07E9 = 2025
│         │     └── Command: 0x1004
│         └── Length: 10 (2+7+1)
└── Header
```

CRC расчёт:
```
0x10^0x04 = 0x14  ^0x07 = 0x13  ^0xE9 = 0xFA  ^0x01 = 0xFB
^0x0F = 0xF4  ^0x0E = 0xFA  ^0x1E = 0xE4  ^0x2D = 0xC9
```

**RX — ACK**:
```

```

**RX — DONE**:
```

```
